About

Jikan is an unofficial PHP/REST API built for parsing data from MyAnimeList.net. It covers the features that the official MyAnimeList API lacks for developers.

It was originally developed as a script for fetching anime related details only and then evolved into an API.

Further more, it's capabilities were extended for developers as it became into a REST based API with it's own domain.


It's called "Jikan", because that means "time" in Japanese. So as a wordplay, Jikan is meant to save developers time from sidetracking their projects to create parsers.


The REST API is built on Lumen and uses Redis as a cache management database.

Jikan is open source and available on Packagist. That is, it can be simply required in PHP projects.

composer require jikan-me/jikan
