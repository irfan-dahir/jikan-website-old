FAQs

General

Are there any limits to the requests?

You can make 5,000 uncached requests per day. Cached requests don't count!

How does Jikan work?

Jikan works by scraping pages from MyAnimeList to parse the data you need.

Is this open source?

Yes! From the source, to the REST API, to this website itself; everything is available on GitHub.

Is this service free?

100%.

I've made a wrapper for Jikan!

That's great! Let me know and I'll add it everywhere.

Features

What differs Jikan from the Official API?

Jikan covers what the official API lacks. It's much simpler to use, requires no authentication and returns data in JSON. :)

What can I get through Jikan?

A whole lot. Refer to the Readme file.
